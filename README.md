## ✨ Возможности

### 🎙 Голосовое управление
- **Google STT** — онлайн распознавание речи (по умолчанию)
- **Vosk STT** — оффлайн распознавание (русская модель, скачивается автоматически)
- Активация по ключевому слову **«Джарвис»**
- Режим диктовки — набирает текст голосом

### 🧠 Искусственный интеллект
- **Yandex AI Studio** — онлайн-мозг (YandexGPT + агенты)
- **Qwen 2.5 (GGUF)** — полностью локальный оффлайн LLM
- **Qwen CLI** — интеграция с Qwen Code
- Память диалогов (настраиваемое количество ходов)

### 🎭 Синтез речи (TTS)
- **Edge TTS** — быстрый облачный синтез (по умолчанию, бесплатно)
- **Qwen3-TTS** — локальное клонирование голоса (GPU)
- **CosyVoice** — клонирование голоса через API или локально
- Кэширование частых фраз для мгновенного отклика

### 🖥 Веб-интерфейс (HUD)
- Анимированный HUD-интерфейс в стиле Iron Man
- Вкладки: Главная, Чат, Команды, Настройки
- Визуализация состояния (слушает / говорит / ожидание)
- Система частиц на фоне

### ⚙ Дополнительно
- 🏠 Умный дом (интеграция с Home Assistant)
- 🔌 Система плагинов
- 🎬 Звуки из фильма Iron Man
- 📸 Анализ экрана (OCR)
- 🎵 Управление медиа (громкость, плей/пауза, треки)
- ⚡ Режим энергосбережения
- 📋 Работа с буфером обмена
- 🎯 Сцены — запуск нескольких действий одной командой

## 📦 Установка

### Требования
- **Windows 10/11**
- **Python 3.10+** ([скачать](https://www.python.org/downloads/))
- **Google Chrome** или **Microsoft Edge** (для веб-интерфейса)
- Микрофон

### Быстрый старт

1. **Скачайте или клонируйте репозиторий:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/jarvis.git
   cd jarvis
   ```

2. **Запустите JARVIS:**
   ```bash
   JARVIS.bat
   ```
   Или напрямую:
   ```bash
   python jarvis_eel.py
   ```

3. **Все зависимости установятся автоматически** при первом запуске:
   - `eel`, `psutil`, `speech_recognition`, `pyautogui`
   - `pyperclip`, `keyboard`, `pygame`, `opencv-python`
   - `pytesseract`, `llama-cpp-python`, `vosk`, `edge-tts`
   - SoX (аудио-утилита) — скачается и установится автоматически

4. **Откроется браузерное окно** с HUD-интерфейсом. Скажите **«Джарвис, привет!»**

## ⚙ Настройка

### Yandex AI Studio (онлайн-мозг)
1. Зарегистрируйтесь на [ai.yandex.cloud](https://ai.yandex.cloud)
2. Создайте API-ключ в AI Studio
3. Скопируйте Folder ID из консоли Yandex Cloud
4. Введите ключ и Folder ID в настройках JARVIS (вкладка «Настройки»)

### Локальный LLM (оффлайн-мозг)
1. Скачайте GGUF-модель (например, [Qwen2.5-1.5B-Instruct](https://huggingface.co/Qwen/Qwen2.5-1.5B-Instruct-GGUF))
2. Поместите файл `.gguf` в папку проекта
3. Укажите путь в настройках или оставьте пустым (модель скачается автоматически)
4. Включите «Локальный мозг» в настройках

### Клонирование голоса (Qwen3-TTS / CosyVoice)
1. Подготовьте аудиофайл с вашим голосом (5-15 секунд, `.mp3` или `.wav`)
2. Поместите его как `voice.mp3` в папку проекта или укажите путь в настройках
3. Включите соответствующий TTS-движок в настройках

## 🎤 Голосовые команды

| Команда | Действие |
|---|---|
| «Джарвис, сколько времени?» | Озвучивает текущее время |
| «Джарвис, открой ютуб» | Открывает YouTube |
| «Джарвис, открой сайт google.com» | Открывает любой сайт |
| «Джарвис, что ты видишь на экране?» | Анализирует экран (OCR) |
| «Джарвис, режим диктовки» | Набирает текст голосом |
| «Джарвис, громче / тише» | Управление громкостью |
| «Джарвис, следующий / предыдущий» | Переключение треков |
| «Джарвис, скриншот» | Делает снимок экрана |
| «Джарвис, что в буфере?» | Читает буфер обмена |
| «Джарвис, молчать / стоп» | Остановка речи |
| «Джарвис, выключись» | Завершение работы (с подтверждением) |
| «Джарвис, [любой вопрос]» | Ответ через ИИ |

### Добавление своих команд
- Через веб-интерфейс: вкладка «Команды» → «Добавить команду»
- Голосом: скажите «режим обучения» и следуйте инструкциям
- Вручную: отредактируйте `jarvis_commands.json`

## ❓ FAQ

**Q: Браузер не открывается**
A: Откройте вручную `http://127.0.0.1:PORT` (порт показывается в консоли)

**Q: Микрофон не работает**
A: Убедитесь, что микрофон выбран как устройство по умолчанию в Windows

**Q: Edge TTS не работает**
A: Требуется интернет-соединение. Edge TTS — облачный сервис

**Q: Локальная модель не загружается**
A: Убедитесь, что у вас достаточно RAM (минимум 4 ГБ для 1.5B модели)
